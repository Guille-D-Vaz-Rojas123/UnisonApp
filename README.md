APP DE LA UNISON INVENTARIO

Es una aplicacion que te permite almacenar una base de datos donde se almacenen almacenes con una cantidad de productos asignada, 
con su nombre, id, almacen, departamento, fecha de creacion, fecha de ultima modificacion y cual fue el ultimo usuario en modificarlo,
estas contienen 3 roles.
-Almacen
-Productos
-Admin
Admin te permitira poder modificar las 2 tablas productos y almacen
Productos te permitira poder modificar solamente la tabla productos y solamente  ver los almacenes como expectador
Almacen te permitira poder modificar solamente la tabla almacen y solamente ver los productos como expectador
